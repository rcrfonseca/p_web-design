Projeto Final COTI Informática

Projeto: Sistema para inventário de máquinas, impressoras e solicitações de reparo dos itens cadastrados.

Autores: Gileno Oliveira Pinto e Raphael Carlos Rocha Fonseca

Objetivos:

  • Manter cadastro de máquinas com suas especificações;
  • Manter cadastro de impressoras com suas especificações;
  • Informar técnicos da área de T.I. sobre problemas das máquinas;

Diferenciais:

  • Menu simples, intuitivo e organizado;
  • Envio de solicitações de reparo para os técnicos diretamente por email através de formulários;

Sobre a programação:

  • Utilizar tecnologia Java através de JSP + Servlets;
  • Orientado a objeto e projetado para utilização online;
  • Forte uso de programação para evitar imagens e respeitando os padrões W3C, beneficiando o usuário na velocidade do sistema;
  • HTML, Tableless e CSS criando boa interface para o usuário;
  • Uso de Ajax para agilizar a busca ao banco de dados;
  • Uso de Jasper Reports para relatórios informando as máquinas já cadastradas;
