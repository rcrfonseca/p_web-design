Projeto Final COTI Informática

Projeto: Site de Informação de Produtos, Lojas e Centros Comerciais de Informática

Autores: Gileno Oliveira Pinto e Raphael Carlos Rocha Fonseca

Objetivos:

  • Informar preços de produtos de informática;
  • Informar lojas de informática;
  • Informar centros comerciais de informática (Infoshoppings);

Diferenciais:

  • Menu simples, intuitivo e organizado;
  • Possibilidade das lojas e Infos serem pesquisados, assim como os produtos;
  • Divisão de setor de atuação das lojas para uma melhor busca ao interesse do usuário;
  • Detalhamento dos serviços oferecidos pela loja;
  • Descrição de uma forma clara de cada periférico vendido de acordo com suas características;
  • Responsabilidade do lojista de cadastrar os modelos e descrição dos produtos;

Sobre a programação:

  • Utilizar tecnologia Java através de JSP + Servlets;
  • Orientado a objeto e projetado utilização online;
  • Forte uso de programação para evitar imagens e respeitando os padrões W3C, beneficiando o usuário na velocidade do site;
  • HTML, Tableless e CSS criando boa interface para o usuário;
  • Uso de Ajax para agilizar a busca ao banco de dados;
  • Uso de Jasper Reports para relatórios informando produtos selecionados;
  • Search Engine Optimization (SEO) para manter uma boa posição nos sites de busca;
